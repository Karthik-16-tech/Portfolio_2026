# Karthik Portfolio

Student developer portfolio built with Vite, React, and TanStack Start.

## Scripts

- `npm run dev` — local development
- `npm run build:vercel` — production build for Vercel (static prerender)
- `npm run preview` — preview production build

## Deploy on Vercel

Import this repo, use framework **Vite**, build command `npm run build:vercel`, output directory `dist/client`.

Optional env: `VITE_SPLINE_URL` (see `.env.example`).
